# Mirror of Maya - Frontend

A beautiful, Maya-themed React frontend for the Near-Duplicate Image Detection system.

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run build
```

3. Make sure the Python backend is running on port 8000:
```bash
python api_server.py
```

## Features

- 🎨 Elegant Maya theme with Sudarshana Chakra aesthetics
- 📊 Real-time metrics dashboard
- 🔍 Query tool for finding duplicates
- 🌌 Galaxy cluster visualization
- 📈 Detailed analytics and charts

## Tech Stack

- React 18 + JavaScript (with SWC)
- Vite
- Recharts
- Lucide React
- Axios
